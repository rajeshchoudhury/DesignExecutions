# DesignExecutions
A production-grade Java 21 implementation for 12 enterprise patterns  — backed by Kafka, PostgreSQL, MongoDB, Redis, and Eureka — showcase Saga orchestration, Event Sourcing, CQRS, Circuit Breaker, Bulkhead, Retry, API Gateway, Service Discovery, Database-per-Service, Sidecar, API Composition, and Strangler Fig migration.
